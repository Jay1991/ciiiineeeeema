class CZekaFP

Remarks

The CZekaFP class provides the user functionality for using Zeka Fiscal Printer device. It includes all of the communications protocol commands that are relative to the end user and some of the basic service commands.

The members documentation specifies the buffer lenght for every command. However it is better to alwais use buffers with length ZFP_MAXBUFLEN to keep compability with future changes (if any).

All commands return an integer error code listed in file ZekaFPdefs.h
