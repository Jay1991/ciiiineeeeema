class CZekaFP

Бележки

Класът CZekaFP съдържа необходимата функционалност за работа със Фискален принтер Зека. Поддържат се всички комуникационни команди които се отнасят към крайния потребител както и някои от основните команди за настройка и сервиз.

Документацията на класа уточнява дължината на буфера за всяка функция член на класа. Препоръчително е винаги, когато е възможно да се ползват буфери с дължина ZFP_MAXBUFLEN с цел запазване на съвместимост с новите версии на класа (в случай че има такива).

Всички команди връщат числова стойност (int) като код на грешка. Тези кодове могат да бъдат открити във файла ZekaFPdefs.h
