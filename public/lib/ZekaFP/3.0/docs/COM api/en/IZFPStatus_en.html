Интерфейс IZFPStatus

Remarks:

The IZFPStatus interface is the result of execution of the following IZekaFP methods: GetStatus()

Properties:

ST0 - Status byte 0

ST1 - Status byte 1

ST2 - Status byte 2

ST3 - Status byte 3

ST4 - Status byte 4


Boolean properties (true/false)

wrongTimer - Wrong clock

wrongDate - Wrong date

wrongRAM - Wrong RAM

clockHardwareError - Clock hardware error

paperOutPrinterError - Paper out or printer error

reportSumOverflow - Report sum overflow

printLogo - Pring logo

nonzeroDailyReport - Non zero daily report

nonzeroArticleReport - Non zero article report

nonzeroOperatorReport - Non zero operator report

duplicateNotPrinted - Duplicate not printed

openOfficialBon - Open official receipt

openFiscalBon - Open fiscal receipt

detailedInfo - Detailed info in the receipt

VATinfo - VAT info in the receipt

missingFiscalMemory - Missing fiscal memory

wrongFiscalMemory - Wrong fiscal memory

fullFiscalMemory - Full fiscal memory

fiscalMemoryLimitNear - Fiscal memory nearly full

hasDecimalPoint - Using decimal point

isFiscal - Printer is fiscal

hasFiscalAndFactoryNum - Fiscal and factory numbers are set

autoCutter - Has auto cutter

hasTransparentDisplay - Has transparent display

baud9600 - Baud rate is 9600 (otherwise 19200)

autoTillOpen - Automatic till open

shortEJMode - Electronic Journal (EJ) print mode (0 = normal, otherwise = short)

EJLimitNear - Electronic Journal (EJ) almost full

fullEJ - Electronic Journal (EJ) full
