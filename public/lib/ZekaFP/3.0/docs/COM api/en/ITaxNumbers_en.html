interface ITaxNumbers

Remarks:

The ITaxNumbers interface is the result of execution of the following IZekaFP methods: GetTaxPercents(), GetDailySums(),

Properties:

taxgrp1 - sum / percents per tax group 'A'

taxgrp2 - sum / percents per tax group 'B'

taxgrp3 - sum / percents per tax group 'C'

taxgrp4 - sum / percents per tax group 'D' (for Romania only)

taxgrp5 - sum / percents per tax group 'E' (for Romania only)

Remarks:

The values of the taxgrp4 and taxgrp5 properties are filled only then you work with Romanian fiscal printer. To choose the Romanian fiscal printer type you need to set the property zfpType = 1 of IZekaFP interface (Bulgarian fiscal printer is setup by default)
