Remarks

The IZekaFP interface provides the user functionality for using Zeka Fiscal Printer device. It includes all of the communications protocol commands that are relative to the end user and some of the basic service commands.

The file zfpcom.dll contains the whole funtionality of IZekaFP and all result interfaces returned from IZekaFP methods

zfpcom.tlb contains the Type library for all interfaces.

All methods return only one result, in cases multiply results are needed, the result is an interface which properties contain all of the returned data. In some cases not all of the result interface properties are filled. In those cases we will enuminate the significant properties in the function documentation. When there is no such enumination this means that the function fill all the properies of the result interface.

All methods of IZekaFP fill the errorCode property. This is how you discover an error.
