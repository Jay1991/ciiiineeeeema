interface IZFPParameters

Remarks:

The IZFPParameters interface is the result of execution of the following IZekaFP methods: GetParameters()

Properties:

fpnum - Fiscal Printer place number

logo - information about logo printing status (0 = not printed, otherwise printed)

till - till status (0 = don't have a till, otherwise till is present)

autocut - auto cutter status (0 = don't have one, otherwise present)

transparent - transparent display status (0 = no display or not transparent, otherwise transparent display)
