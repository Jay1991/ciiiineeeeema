interface IDailyReport

Remarks:

The IDailyReport interface is the result of execution of the following IZekaFP methods: GetDailyReport(), GetOperatorCounter()

Properties:

counter - report counter

lastreport - number of the last fiscal report

ecl - electronic cash line number (if present)

datetime - day of the month, month, year, hour and minute of the last report
