interface IPayTypes

Remarks:

The IPayTypes interface is the result of execution of the following IZekaFP methods: GetPayTypes(), GetDailyPayments(), GetDailyIncomes(), GetDailyOfficialIncomes(), GetDailyOfficialExpenses(), GetOperatorIncomes(), GetOperatorExpenses(), GetOperatorReceives()

Properties:

paytype0 - pay type 0 name
paytype1 - pay type 1 name
paytype2 - pay type 2 name
paytype3 - pay type 3 name
paytype4 - pay type 4 name

paytype0sum - sum for pay type 0
paytype1sum - sum for pay type 1
paytype2sum - sum for pay type 2
paytype3sum - sum for pay type 3
paytype4sum - sum for pay type 4

operations - total number of operations
