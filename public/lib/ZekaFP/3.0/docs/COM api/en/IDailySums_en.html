interface IDailySums

Remarks:

The IDailySums interface is the result of execution of the following IZekaFP methods: GetDailyInfo(), GetOperatorInfo()

Properties:

clients - number of the clients

discounts - number of the discounts
sumdiscount - total sum for daily discounts

adding - number of the additions
sumadding - total sum for daily additions

voids - number of the voids
sumvoids - total sum for daily voids
