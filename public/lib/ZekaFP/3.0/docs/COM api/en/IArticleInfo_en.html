interface IArticleInfo

Remarks:

The IArticleInfo interface is the result of execution of the following IZekaFP methods: GetArticleInfo()

Properties:

name - article name

price - price

taxgroup - tax group

turnover - turnover

sells - sells count

counter - number of the last report

reportdate - day of the month, month, year, hour and minute of the last report

department - department number
