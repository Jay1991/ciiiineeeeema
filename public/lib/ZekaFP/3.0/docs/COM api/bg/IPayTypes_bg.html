Интерфейс IPayTypes

Бележки

Интерфейсът IPayTypes се връща като резултат от изпълнението на функциите: GetPayTypes(), GetDailyPayments(), GetDailyIncomes(), GetDailyOfficialIncomes(), GetDailyOfficialExpenses(), GetOperatorIncomes(), GetOperatorExpenses(), GetOperatorReceives()

Свойства (properties)

paytype0 - име на тип плащане 0
paytype1 - име на тип плащане 1
paytype2 - име на тип плащане 2
paytype3 - име на тип плащане 3
paytype4 - име на тип плащане 4

paytype0sum - сума натрупана по тип плащане 0
paytype1sum - сума натрупана по тип плащане 1
paytype2sum - сума натрупана по тип плащане 2
paytype3sum - сума натрупана по тип плащане 3
paytype4sum - сума натрупана по тип плащане 4

operations - брой на операциите
exchRate - валутен курс (за тип плащане 4)
