Интерфейс IArticleInfo

Бележки

Интерфейсът IArticleInfo се връща като резултат от изпълнението на функциите: GetArticleInfo()

Свойства (properties)

name - име на артикула

price - цена

taxgroup - данъчна група

turnover - оборот

sells - данъчна група

counter - брояч на последния отчет

reportdate - ден, месец, година, час и минута на последния отчет

department - номер на департамент

quantity - продадено количество
