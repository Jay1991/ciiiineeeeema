Интерфейс IDailyReport

Бележки

Интерфейсът IDailyReport се връща като резултат от изпълнението на функциите: GetDailyReport(), GetOperatorCounter()

Свойства (properties)

counter - брояч на отчета

lastreport - номер на последния отчет във ФП

ecl - номер на електронна контролна лента (ако има такава)

datetime - дата на последния отчет
