Интерфейс IGetCurrentBonInfoRes

Бележки:

Интерфейсът IGetCurrentBonInfoRes се връща като резултат от изпълнението на функцията GetCurrentBonInfo()

Свойства (properties)

flags - проверявайте стойността на това свойство. Ако е различна от нула има отворен бон и стойностите на останалите свойства са валидни.

purchases - брой продажби натрупани в бона.

taxgrp1 - сума натрупана в данъчна група 'А' ('A')

taxgrp2 - сума натрупана в данъчна група 'Б' ('B')

taxgrp3 - сума натрупана в данъчна група 'В' ('C')

taxgrp4 - сума натрупана в данъчна група 'Г' ('D')

taxgrp5 - сума натрупана в данъчна група 'Д' ('E')

taxgrp6 - сума натрупана в данъчна група 'Е' ('F')

taxgrp7 - сума натрупана в данъчна група 'Ж' ('G')

taxgrp8 - сума натрупана в данъчна група 'З' ('H')

novoid - true ако е забранено войдиранено, иначе false.

dds - true ако се печата ДДС, иначе false

invoice - true ако бона е фактура, иначе false

powerdown - true ако е имало спад на напрежение в бона, иначе false

detailed - true ако се печатат детайли в бона, иначе false

paystarted - true ако е започнало плащане на бона, иначе false

paid - true ако плащането е завършило, иначе false

change - ресто от последния фискален бон

changeType - тип плащане на рестото от последния фискален бон

Бележки:

Стойностите на taxgrp4 и taxgrp5 се попълват само когато се работи с Румънски вариант на фискалния принтер. За да изберете Румънски вариант задайте стойност на свойството zfpType = 1 на интерфейса IZekaFP; (по подразбиране е български вариант)
