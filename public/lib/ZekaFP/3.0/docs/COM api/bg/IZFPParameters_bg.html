Интерфейс IZFPParameters

Бележки

Интерфейсът IZFPParameters се връща като резултат от изпълнението на функциите: GetParameters()

Свойства (properties)

fpnum - номер на касово място

logo - дали се печата логото (0 = не се печата)

till - дали има чекмедже (0 = няма)

autocut - дали има автоматичен резач (0 = няма)

transparent - дали външния дисплей е прозрачен (0 = не е прозрачен)

shortEJ - съкратена електронна контролна лента - ЕКЛ (0 = подробна)

currency - позлване на валута в бона (0 = не се ползва)
